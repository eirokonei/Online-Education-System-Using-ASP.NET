### `How to Run`

1. Download the project and open Online Learning.sln in Visual Studio.
2. Open SQL Server Management Studio.
3. Attach ELearningDB.mdf file in Database.
4. Open HomeController.cs
5. Press CTRL+F5

Finally you got it!!
